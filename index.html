Work 01

Immersive Real Estate Experience for Primula Costruzioni

Real-time architectural visualization software developed in Unreal Engine 5 to support the sales and presentation of a new residential building within its surrounding context. The project enables interactive exploration of the architecture and interior spaces through real-time rendering, dynamic natural lighting, and a touchscreen-first interface.

Role

3D visualization, environment assembly, interface design, Blueprint development, real-time presentation.

Tools

  • Unreal Engine
  • Blender
  • Photoshop
  • Illustrator
  • Cesium
  • ComfyUI

Why it matters

A sales tool, not just a render.

The experience was designed to make the project legible and desirable in real time. Instead of static imagery only, the interface supports active navigation, daylight variation, and spatial understanding during client presentations.

Workflow

From modeled context to packaged executable.

01

Building and context modeling

The architectural intervention and surrounding district were modeled in Blender to create a clean, controllable base for real-time visualization.

02

Unreal assembly and geospatial integration

The project was assembled in Unreal, enriched with Cesium context data, materials, lighting, furnishing, and atmosphere to improve readability and immersion.

03

Interactive systems and packaging

Blueprint logic, widgets, GameMode behavior, and controller systems were developed for touchscreen interaction, then packaged into a standalone Windows application.

Work 03

“CAPO” from “Necroville” — 2D to Real-Time 3D

A 3D adaptation of “Capo” from the 2D series Necroville, developed through a game and animation-oriented workflow. The project focuses on readable silhouette, deformation-friendly topology, stylized surface treatment, rig preparation, and motion testing.

Role

Character modeling, topology cleanup, material lookdev, lighting, rig preparation, animation tests.

Tools

  • Blender
  • ComfyUI
  • Rigify

Focus

From 2D identity to a presentable 3D character.

The project explores how a stylized illustrated character can be translated into a real-time 3D asset while preserving silhouette, appeal, and production logic. The goal was to create a model that feels faithful to the original design but also functional for rigging, lighting, and animation tests.

Workflow

Reference, model, shade, rig, animate.

01

Reference setup

Turnaround poses provided by the concept artist were expanded with ComfyUI-generated front and side T-pose references to improve consistency during modeling.

02

Modeling and look development

The character was modeled with a focus on stylized proportions, readable silhouette, and materials that support the final presentation.

03

Rigging and animation tests

An armature and Rigify control rig were assembled to test deformation, posing, and basic animated performance directly in Blender.

Work 04

Character Sculpt Study

Stylized character sculpt in development, focused on anatomy, proportion, silhouette, and a future transition toward animation-ready topology and Unreal-based control rig workflows.

Status

Lab / Work in Progress

Focus

Stylized character sculpt in development, focused on anatomy, proportion, silhouette, and a future transition toward animation-ready topology and Unreal-based control rig workflows.

Current direction

Stylized sculpting with a future real-time pipeline in mind.

The study is centered on anatomy, proportional clarity, silhouette, and the transition from sculpt exploration to animation-ready topology. The goal is not only sculpt fidelity, but a stronger bridge toward rigging and real-time presentation.

Workflow

From 2D reference to a future game-ready character.

01

Base modeling from 2D reference

The project begins from 2D reference artwork used to define proportions, silhouette, and primary forms, building a clean base model before the sculpting stage.

02

Sculpting, detail, and retopology

The character is being refined through digital sculpting, focusing on secondary forms, surface definition, and stylized anatomy, while retopology is planned to create a cleaner animation-ready mesh.

03

Unreal Control Rig and IK setup

The final objective is to bring the character into Unreal Engine and build an inverse kinematics Control Rig designed for game-oriented animation, posing, and character control.

About
